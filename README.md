## About The Project
Resuta.ai is a Website that shows all of the matches of brazilian CS:GO teams that will happen. The project has two pages, the principle, where you can find all of the matches coming up, and the match page, where you can find all the information about a specific match.

### Built With

* NextJs
* Next API
* TypeScript


<!-- GETTING STARTED -->
## What's Inside?

This is a Full-Stack project made in order to learn NextJS and TypeScript. 

I Used the Next API to build the Back-End and the ServerSideProps to fetch all the information to the page.
